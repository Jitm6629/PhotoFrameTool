# PhotosBorder 摄影照片白边排版工具
一款本地离线 Python 桌面GUI工具，批量为摄影作品添加自定义白边，自动读取照片EXIF拍摄参数，底部左右分区排版相机LOGO与拍摄信息，适用于摄影作品集、社交平台配图、线下打印输出。

## 🎯 核心优势
1. **完全离线本地运行**：所有图片处理在本机完成，不上传原图，保护摄影隐私，支持超大体积高清照片批量处理
2. **四边独立白边调节**：上/下/左/右边距可单独自定义像素，自由控制画面留白比例
3. **EXIF自动解析**：一键提取相机、镜头、光圈、快门、ISO、焦距；支持手动修改参数覆盖原图信息
4. **底部双栏专业排版**
   - 左侧：垂直居中相机LOGO区域，支持透明PNG素材（佳能/尼康/索尼等）
   - 右侧：紧贴右边缘标准化参数展示 `快门速度 ISO xxx f/xx 相机型号`
5. **轻量化无捆绑**：基于Python内置tkinter开发，无需复杂环境，导出高清JPG保留画质
6. **批量高速处理**：一次性导入数十张原图，一键批量生成成品图

## 🛠 技术栈
- GUI界面：tkinter（Python内置，无需额外安装）
- 图像处理：Pillow (PIL) 画布合成、文字渲染、图片缩放
- EXIF元数据解析：piexif
- 适配环境：Python 3.10 / 3.11 Windows
